# 准备工作 {#prepare}

## 1. 下载并安装NET.10

### 1.1 下载NET.10
访问[微软官方地址](https://builds.dotnet.microsoft.com/dotnet/WindowsDesktop/10.0.9/windowsdesktop-runtime-10.0.9-win-x64.exe)进行下载

### 1.2 安装NET.10
双击下载下来的文件，点击`安装`。  
![](/image/v3/NET10install.png)
如果你双击打开NET.10的安装程序是这样的，这表明你已经有了基础的环境，直接点击 `关闭`即可。
![](/image/v3/NET10install2.png)

# 安装软件

## 2. 下载安装程序
1.1 前往[CNB(中国镜像仓库)](https://cnb.cool/ETS2LA-CN/Euro-Truck-Simulator-2-Lane-Assist/-/releases/latest)下载 这里会自动跳转至最新的版本 不管哪个版本 下最新版本中我红框框的地方就行了
![](/image/v3/CNB%20download.png)

::: details GitHub(官方下载仓库)
前往[GitHub(官方下载仓库)](https://github.com/ETS2LA/ETS2LA/releases/latest)下载 这里会自动跳转至最新的版本 不管哪个版本 下最新版本中我红框框的地方就行了
![](/image/v3/GitHub%20download.png)
:::

::: tip 小提示
推荐CNB下载，github也可使用

点击蓝字即可跳转至网站

注意：无论你从何处下载安装，更新时只会检索下载安装时选择的平台
:::

### 2.1. 运行安装程序
双击运行刚才下载的安装程序，可能会出现`Windows defender`的提示，直接点击`更多信息`，展开之后选择`仍要运行`即可。
![](https://tc.ets2la.cn/25-12/20250721084016.png)  
然后会进入到安装向导，点击`下一步`即可  
![](/image/v3/ETS2LA%20install1.png)
勾选`我接受许可协议中的条款`并点击`下一步`  
![](/image/v3/ETS2LA%20install2.png)
点击`安装`以来安装软件
![](/image/v3/ETS2LA%20install3.png)
这一步请慢慢等待 应该很快会安装成功
![](/image/v3/ETS2LA%20install4.png)
安装完成后默认勾选`启动ETS2LA`，然后点击完成
![](/image/v3/ETS2LA%20install5.png)
